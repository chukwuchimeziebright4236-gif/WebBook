What is HTML?

HTML stands for HyperText Markup Language. It is the standard backbone used to create and design documents on the World Wide Web.

Think of it this way: If a website were a house, HTML is the structural framing and walls. CSS would be the paint and interior design, and JavaScript would be the electricity and plumbing that makes things move and function.

  • HyperText: Refers to the way web pages are linked together. The hyperlinks you click on are the "hypertext."
  • Markup Language: Means it uses tags to annotate (or "mark up") text, images, and other content so a web browser knows how to display them.

The Anatomy of an HTML Element

HTML uses elements to structure content. An element is usually made up of a start tag, some content, and an end tag.

Here is a quick breakdown of a standard element:

<p>This is a paragraph of text.</p>
  • <p> (Start Tag): Tells the browser where the paragraph begins.
  • This is a paragraph of text (Content): The actual text or media being displayed.
  • </p> (End Tag): Tells the browser where the paragraph ends (notice the forward slash /).

Structure of a Basic HTML Document

Every standard HTML page follows a specific boilerplate structure. Here is what the bare minimum looks like:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First Web Page</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is my very first paragraph written in HTML!</p>
</body>
</html>

Breaking Down the Blueprint:

  • <!DOCTYPE html>: Defines the document type and tells the browser to expect modern HTML5.
  • <html>: The "root" element that wraps all the content on the entire page.
  • <head>: Contains behind-the-scenes information (metadata) about the page, like the character encoding and the title that appears in your browser tab.
  • <title>: Sets the name of the page shown on the browser tab.
  • <body>: Contains everything that is actually visible to the user on the screen (text, images, buttons, videos).

Common HTML Tags You'll Use Most Often

When you are starting out, you will find yourself using a handful of core tags repeatedly:

Tag Name What it Does
<h1> to <h6> Headings Creates titles and subtitles. <h1> is the largest/most important; <h6> is the smallest.
<p> Paragraph Groups normal blocks of text.
<a> Anchor (Link) Creates hyperlinks to other pages using the href attribute (e.g., <a href="https://google.com">Google</a>).
<img> Image Embeds an image. It is "self-closing" and uses src for the image path (e.g., <img src="logo.png" alt="Logo">).
<ul> / <ol> Lists <ul> creates bulleted lists; <ol> creates numbered lists. Both use <li> (List Item) tags inside them.
<div> Division A generic container used to group elements together for styling or layout purposes.

Ready to Test It Out?

You don't need any fancy software to start writing HTML.

  1. Open up a basic text editor like Notepad (Windows) or TextEdit (Mac).
  2. Copy and paste the "Basic HTML Document" code block from above.
  3. Save the file as index.html (make sure it ends in .html, not .txt).
  4. Double-click that saved file, and it will open up right in your web browser!
